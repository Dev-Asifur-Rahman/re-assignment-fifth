Important FAQ

What is the Document Object Model (DOM)?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a webpage as a tree of nodes, which can be manipulated using JavaScript.

How do you select an element from the DOM?

To select an element in an accordion, you can use document.querySelector() for the first matched element or document.querySelectorAll() for all matching elements. Then, you can manipulate the accordion items, such as toggling their visibility.

What is event delegation in the context of DOM,and why is it useful?

Event delegation attaches one event listener to a parent element, which handles events for child elements, even if they are added later. It's useful for better performance and easier event management.

How do you manipulate an elements attributes and styles using the DOM?

To manipulate an element's attributes, you can use setAttribute('attr', 'value') to change them. To manipulate styles, you modify the style.property = 'value' for inline styles.
